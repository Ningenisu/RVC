# リアルタイム音声認識・合成アプリケーション

Node.jsとJavaScriptを使用した、マイクから音声をリアルタイムで認識し、テキスト化して機械音声で読み上げるWebアプリケーションです。

## 機能

- 🎤 **リアルタイム音声認識**: マイクから入力された音声をリアルタイムでテキスト化
- 📝 **テキスト表示**: 認識されたテキストをリアルタイムで表示
- 🔊 **音声合成**: 認識されたテキストを機械音声で読み上げ
- ⚙️ **音声設定**: 音声の種類、速度、高さ、音量を調整可能

## 技術スタック

- **Node.js**: Express.jsを使用したWebサーバー
- **HTML/CSS/JavaScript**: フロントエンド
- **Web Speech API**: ブラウザの音声認識（SpeechRecognition）と音声合成（SpeechSynthesis）API

## 必要な環境

- Node.js 14以上
- Chrome、Edge、OperaなどのChromiumベースのブラウザ（Web Speech APIをサポート）

## インストール方法

1. 依存パッケージをインストール:
```bash
npm install
```

## 起動方法

### 基本的な起動方法（全OS共通）

```bash
npm start
```

ブラウザで `http://localhost:3000` にアクセスしてください。

### 便利な起動スクリプト

#### Windowsの場合
```bash
start.bat
```

#### macOS/Linuxの場合
```bash
chmod +x start.sh
./start.sh
```

スクリプトを使用すると、依存関係の自動インストールとブラウザの自動起動が行われます。

## 使い方

1. **録音開始**: 「録音開始」ボタンをクリックしてマイクへのアクセスを許可
2. **音声入力**: マイクに向かって話すと、リアルタイムでテキスト化されます
3. **録音停止**: 「録音停止」ボタンで音声認識を停止
4. **読み上げ**: 「読み上げ」ボタンで認識されたテキストを音声で読み上げ
5. **設定調整**: 音声の種類、速度、高さ、音量をスライダーで調整可能

## 注意事項

- このアプリケーションはWeb Speech APIを使用しているため、Chrome、Edge、OperaなどのChromiumベースのブラウザでのみ動作します
- 初回使用時はブラウザからマイクへのアクセス許可が必要です
- HTTPS接続またはlocalhostでのみ動作します（マイクアクセスのセキュリティ要件）
